OBJETIVO

Essa aplicação foi criada com o intuido de fazer o Web Scraping da pagina www.infomoney.com.br/cotacoes/petrobras-petr4/historico,
retirando as informações em tempo real dos dados das ações da petrobras para que os mesmos sejam utilizados dentro de um
modelo de Inteligência Artificial com o intuito de prever os valores de fechamento.

BIBLIOTECAS

Neste projeto foram usados as bibliotecas:

•BeatifulSoup para fazer o WebScraping extraindo dados de paginas HTML
•Requests para fazer as requisições HTTP
•Pandas para fazer as manipulações
•Sklearn para o aprendizado de máquina 

É necessário a instalação das bibliotecas
